# PWA Icons for StayGo

## Generating Icons

You have two options to generate the PWA icons:

### Option 1: Use the HTML Generator (Recommended)
1. Open `generate-icons.html` in your web browser
2. Right-click on each generated icon and save them to this directory
3. Icons will be automatically named correctly

### Option 2: Use Your Own Icons
Replace these placeholder files with your custom hotel-themed icons:

**Required icons:**
- `pwa-192x192.png` - 192x192 pixels (required for Android)
- `pwa-512x512.png` - 512x512 pixels (required for Android)
- `apple-touch-icon.png` - 180x180 pixels (required for iOS)
- `favicon.ico` or `favicon.png` - 32x32 pixels

**Recommended additional icons:**
- `pwa-144x144.png` - 144x144 pixels
- `pwa-256x256.png` - 256x256 pixels
- `pwa-384x384.png` - 384x384 pixels

## Icon Design Guidelines

- Use solid background color (transparency may cause issues on some devices)
- Use the StayGo brand color #0E64D2 as primary color
- Ensure icons are recognizable at small sizes
- Consider creating maskable icons for Android adaptive icons
- Test icons on both light and dark backgrounds

## Current Status

Placeholder SVG icon (`icon.svg`) is available. Generate PNG versions using the HTML generator or create custom icons following the guidelines above.
